## üñºÔ∏è How to Add Images

### Method 1: Google Drive (Recommended)

1. **Upload image to Google Drive**
2. **Right-click** ‚Üí **Get link**
3. **Set to:** "Anyone with the link" ‚Üí **Viewer**
4. **Copy the File ID** from URL:
   ```
   https://drive.google.com/file/d/FILE_ID_HERE/view
   ```
5. **Convert to direct link:**
   ```
   https://drive.google.com/uc?export=view&id=FILE_ID_HERE
   ```
6. **Paste into Google Sheet**

### Method 2: Imgur (Easiest)

1. Go to https://imgur.com/upload
2. Upload image
3. Right-click image ‚Üí **Copy image address**
4. Paste URL into Google Sheet

### Method 3: Cloudinary (Professional)

1. Upload to Cloudinary
2. Copy **Secure URL**
3. Paste into Google Sheet

## üí° Pro Tips

1. **Use consistent naming:** camelCase for keys (e.g., `heroImage`, not `hero_image`)
2. **Keep it simple:** Don't overcomplicate - start with basic fields
3. **Test frequently:** Check API endpoint after each major change
4. **Backup regularly:** File ‚Üí Make a copy (weekly backups)
5. **Version control:** Use Google Sheets version history
6. **Multiple admins:** Share edit access with trusted team members only
